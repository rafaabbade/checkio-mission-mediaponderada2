Enunciado

Fernanda, coordenadora de um curso, precisa calcular médias ponderadas das notas dos alunos para finalizar os registros de desempenho. Ela requer uma função que possa receber quaisquer notas e seus respectivos pesos, e calcular a média ponderada.

Entrada: Quatro números float: a primeira nota, seu peso, a segunda nota e seu peso.

Saída: Um número float indicando a média ponderada das notas.

countConsecutiveSummers(42) == 4
countConsecutiveSummers(99) == 6
{% else %} 
media_ponderada(7.5, 3, 8, 7) == 7.85
media_ponderada(10, 5, 6, 5) == 8.0
media_ponderada(4.5, 2, 5, 3) == 4.8
{% endif %}
